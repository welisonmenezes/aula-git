CRUD tags
-------------------------------------------------------------------------------

git tag -a tag-name-here -m 'message tag here'
	- Cria uma tag nova no commit corrent. -a cria a tag anotada

git tag tag-name-here -m 'message tag here'
	- Cria uma tag não anotada no commit corrent.

git tag -a tag-name-here commit-id-here
	- Cria uma tag para um dado commit

git tag
	- Lista todas as tags

git show tag-name-here
	- Mostra detalhes da tag especificada

git push remote-name-here tag-name-here
	- Publica a tag no repositório remoto

git push remote-name-here --tags
	- Publica todas as tags no repositório remoto

git tag --delete tag-name-here
	- Deleta uma dada tag do repositório

git push --delete remote-name-here tag-name-here
	- Deleta uma dada tag do repositório remoto

git push remote-name-here :tag-name-here
	- Equivalente ao comando acima





CRUD stash
-------------------------------------------------------------------------------
	
git stash
	- Tira o estado sujo do seu diretório de trabalho e o salva uma pilha de pendências

git stash save "Your message here"
	- Cria um stash com uma mensagem amigável

git stash list
	- Lista suas pilhas de pendências existentes

git stash apply
	- Aplica a pilha de pendências na branch corrente. Irá aplicar o stash mais recente

git stash apply stash-name-here
	- Aplica o dado stash à branch corrente

git stash pop
	- Aplica a pilha de pendências na branch corrente e em seguida o deleta

git stash drop stash-name-here
	- Deleta o dado stash

git stash branch branch-name-here
	- Recupera um stash e cria uma branch a partir dela
