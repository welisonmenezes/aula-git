Desfazendo/Alterando commit
-------------------------------------------------------------------------------

git reset commit-id-here
	- Reseta para um commit específico. *Deleta os posteriores

git reset HEAD~1
	- Retorna para um commit anterior ao atual. *Retorna 1 a cada execução.

git reset --hard HEAD~1
	- Retorna para um commit anterior ao atual e remove do stage. *Retorna 1 a cada execução.

git checkout commit-id-here
	- Cria e vai para uma branch com o estado do commit especificado

git checkout HEAD~1
	- Retorna ao estado anterior do commit atual numa nova branch. *Retorna 1 a cada execução.

git reflog
	- Lista todas as branchs, inclusive as geradas por git checkout HEAD~1

git revert commit-id-here
	- Reverte as alterações do commit especificado. *Cria um novo commit

git commit --amend
	- Permite adicionar novas alterações no último commit

git commit --amend -m "Edited message"
	- Permite adicionar novas alterações no último commit com uma nova mensagem


Desfazendo/Alterando working are and stage area
-------------------------------------------------------------------------------

git checkout file-name-here
	- Remove as alterações do arquivo especificado que ainda não foram adicionadas ao stage

git checkout HEAD file-name-here
	- Remove do stage e do working area

git reset
	- Remove as alterações adicinadas no stage e q não foram commitadas

git reset --hard
	- Remove as alterações adicinadas no stage e também da área de trabalho
